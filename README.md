# ball_game

Simple and pretty fun ball game for two players made with Python and PyGame.

Ball Game special features:

* Game for two players.
* Shutter control via keyboard.
* Players' score display.
* Adding blocks and extra balls.
* Special effects by balls. 
* Reset balls feature.

Project is based on a demo game and a series of articles presented here:
https://gitlab.com/the-gigi/pygame-breakout

## Getting Started and Playing

To play this game locally you should have:

* Game folder copy.
* Python 3.x.
* PyGame.

Play game with your friend by running: 
```
python ball_game.py
```
And press "PLAY" button.

## Rules and Controls

### Game Goal

The main goal of the Ball Game is to get less number of goals than your opponent.
Each time the ball hits one of your gates goal happens. Use shutter to protect your gates from ball hits.
The player who gets 10 goals to his/her own gates loses game.

There are some "magic" balls in this game, so don't panic if something tricky happens.

### Controls

There are two players in "Ball Game". 
Player 1 (or left player) can move his/her shutter using "W" and "S" buttons on the keyboard.
Player 2 (or right player) can move his/her shutter using "UP" and "DOWN" buttons.

To start game press "PLAY" button. To exit game press "QUIT".
You may add some blocks and extra balls to make your game more complicated by press "ADD BLOCKS" and "ADD BALL" respectively.
To restart all balls use "RESTART BALLS" button.
